# Guess My Number

A simple browser-based game where the player attempts to guess a randomly generated number between 1 and 20.

## 🎮 How to Play

1. Open `index.html` in a web browser.
2. A secret number between **1** and **20** is chosen automatically.
3. Enter your guess in the input box and click the **Check!** button.
4. The game will tell you if your guess is too low, too high, or correct.
5. Your score starts at **20** and decreases by 1 with each incorrect guess.
6. When you guess correctly, the background color changes to green and the hidden number is revealed.
7. The highest score you've achieved during the session is tracked and displayed.
8. Click the **Again!** button at any time to reset the game and try again.

## 🛠️ Files

- `index.html` – The main HTML markup for the game.
- `style.css` – Styles and layout for the game interface.
- `script.js` – JavaScript logic handling game state, user input, and UI updates.

## 💡 Features

- Random number generation on each round
- Dynamic messages for feedback (too low, too high, correct, or no input)
- Score tracking and high‑score persistence during the session
- Reset functionality without page reload
- Simple responsive layout

## 📦 Requirements

All you need is a modern web browser; no server or build tools are required.
